If you just provide sender email and password and receiver mail address in scrap.py code and run the script then it will send mail every 1 hours with if it is found the text or not untill you stop it.
1. To run this code u need to go to Web_Scrapping folder from command line
2. Create virtualenvironment inside it by 
	virtualenv .
3. Then activate it by
	 source bin/activete
4. Install requests by
	pip install requests
5. Install  BeautifulSoup4 by
	pip install BeautifulSoup4
6. Finally run scrap.py by
	python scrap.py
